Methods

(static) polyfill(parentopt, polyfillConfiguration_opt) → {function}

Initializes the inputs and returns a function to re-position the option lists found in the container

Parameters:
Name Type Attributes Default Description
parent HTMLElement <optional>
 document.body

a DOM element which contains the targeted form inputs.
polyfillConfiguration_ DataListJS.PolyfillOptions <optional>

a configuration object used to customize the css of the generated DOM elements which make up the polyfill
Returns:

A function which updates the positions of the generated compensatory select boxes according to the positions of the input elements. The implementation should call this function when the web page layout has changed the positions of the form inputs.

Type
function
Example
if (DataListJS.isNotNativelySupported){
   updatePositions = DataListJS.polyfill(
      document.body,
      {
         uListStyle: {
            cssClassName: 'datalist-polyfill-demo',
            styles: { maxHeight: "120px" }
         }
      }
   );
}
